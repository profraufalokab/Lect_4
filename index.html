شرح خصائص CSS الأساسية

دليل عملي لتنسيق صفحات الويب

1. الخلفية (background-color & background-image)

تُستخدم هذه الخصائص لتحديد لون أو صورة خلفية للعناصر.

الخاصية الوصف القيم الممكنة
background-color تحديد لون الخلفية اسم اللون، HEX، RGB
background-image إضافة صورة كخلفية url('path') 

div {
    background-color: lightblue;
    background-image: url('pattern.png');
    background-size: cover;
}
هذا نص بخلفية ملونة + صورة.

2. النصوص (Text)

خصائص النصوص تتحكم بمحاذاة النص، تزيينه، وتحويله.

الخاصية الوصف القيم
text-align محاذاة النص left, right, center, justify
text-decoration تزيين النص none, underline, overline, line-through
text-transform تحويل حالة الحروف uppercase, lowercase, capitalize 

p {
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
}
هذا نص مُحاذى للوسط ومسطّر

3. الخطوط (Font)

خصائص الخطوط تحدد نوع الخط، حجمه، ووزنه.

الخاصية الوصف القيم
font-family عائلة الخط Arial, Tahoma, sans-serif...
font-size حجم الخط px, em, rem, %
font-weight وزن الخط normal, bold, 100-900 

h1 {
    font-family: 'Tahoma', sans-serif;
    font-size: 24px;
    font-weight: bold;
}
هذا نص بخط Tahoma وحجم 24px

4. اللون (Color)

خاصية color تحدد لون النص.


p {
    color: red;
}
هذا نص لونه أحمر

5. الحدود (Border)

تضيف إطاراً حول العنصر.

الخاصية الوصف القيم
border-width سمك الحدود px
border-style نمط الحدود solid, dashed, dotted...
border-color لون الحدود اسم لون، HEX، RGB 

div {
    border: 2px solid black;
}
عنصر بحد أزرق متقطع

6. الحشو الداخلي (Padding)

المسافة بين محتوى العنصر وحدوده الداخلية.


div {
    padding: 20px;
}
نص مع مسافة داخلية (Padding) 20px

7. الهوامش الخارجية (Margin)

المسافة بين العنصر والعناصر المجاورة.


div {
    margin: 30px;
}
عنصر مع هامش خارجي 30px
